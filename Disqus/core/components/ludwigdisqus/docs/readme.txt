--------------------
LudwigDisqus
--------------------
Version: 1.0.0
Since: October 12th, 2014
Author: Thomas Ludwig <thomas@ludwig.im>

LudwigDisqus integrates Disqus to your Modx installation. 
The comments are imported regularly into your database by crontab, so that you can use it for search engine optimization.

Visit https://ludwig.im/en/projects/disqus-ludwigdisqus-for-modx for more information
