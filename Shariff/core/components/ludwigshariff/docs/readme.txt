--------------------
LudwigShariff
--------------------
Version: 1.0.0
Since: December 1st, 2014
Author: Thomas Ludwig <thomas@ludwig.im>

LudwigShariff integrates Shariff to your Modx installation.
Facebook, Google+ and Twitter supply official sharing code snippets which quietly siphon personal data from all page visitors.
Shariff (/ˈʃɛɹɪf/) enables visitors to see how popular your page is on Facebook and share your content with others without needless data leaks.
Shariff is an open-source, low-maintenance, high-privacy solution maintained by German computer magazine c't and heise online.

Visit https://ludwig.im/en/projects/ludwigshariff-shariff-for-modx for more information
